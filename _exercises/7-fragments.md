# 7 – GraphQL Fragments

## Task 1 – Create GraphQL fragments

Create a `User` GraphQL fragment called `UserBaseInfo` with following fields:

- `id`
- `username`
- `displayName`
- `photo`

Reuse the fragment on all queries & mutations where possible.
